# 🍽️ Persian Cuisine

Persian Cuisine is a React Single Page Application (SPA) that showcases traditional Iranian dishes.
Users can explore categories such as main dishes, desserts, and drinks, and view detailed recipes and ingredients.

---

## 🌐 Live Demo

👉 https://persian-cuisine.vercel.app/

---

## ✨ Features

* 🔁 State-based navigation (no routing)
* 📄 Multiple pages (Home, Main Dishes, Desserts, Drinks)
* 🍲 Detailed food view with ingredients and recipes
* 🧩 Component-based architecture
* 🔄 Dynamic rendering using map()
* 🎨 Responsive design
* 🖼️ Background image support

---

## 🛠️ Technologies Used

* React
* JavaScript (ES6)
* CSS Modules

---

## 🧠 How It Works

The application uses React state to simulate page navigation instead of routing.
Each category displays a list of foods, and selecting a food shows its full details.
